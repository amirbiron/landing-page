# BotPro - דף נחיתה לבוט שירות לקוחות חכם

## 📋 תיאור הפרויקט

דף נחיתה מודרני ומרשים עבור **BotPro** - צ'אטבוט שירות לקוחות חכם לעסקים, מבוסס בינה מלאכותית, שרץ בטלגרם ומגיע עם פאנל ניהול אינטרנטי.

## ✨ תכונות עיקריות

### עיצוב ותצוגה
- 🎨 **עיצוב מודרני** - Dark mode עם גרדיאנטים כחול-סגול
- 📱 **רספונסיבי מלא** - מותאם למובייל, טאבלט ודסקטופ
- 🌐 **תמיכה מלאה בעברית** - RTL נכון עם גופן Assistant
- ✨ **אנימציות חלקות** - Fade in, slide up, ואפקטים חזותיים

### סקשנים בדף

1. **Hero Section** - כותרת ראשית עם CTA בולט ומוקאפ של שיחה בטלגרם
2. **מה הלקוחות שלך מקבלים** - 4 קארדים עם פיצ'רים ללקוח הסופי
3. **מה בעל העסק מקבל** - 6 יתרונות לבעל העסק
4. **איך זה עובד** - 3 שלבים פשוטים להפעלה
5. **טכנולוגיה** - המחסנית הטכנולוגית (GPT-4, FAISS, טלגרם)
6. **המליצות** - 3 ציטוטים מלקוחות מרוצים
7. **מחירים** - 3 תוכניות (בסיסי, עסקי, פרמיום)
8. **CTA סופי** - קריאה לפעולה עם כפתור בולט
9. **Footer** - פרטי יצירת קשר, קישורים ורשתות חברתיות

## 🛠️ טכנולוגיות בשימוש

- **HTML5** - מבנה סמנטי
- **CSS3** - עיצוב מתקדם עם CSS Variables, Grid, Flexbox
- **JavaScript (Vanilla)** - אינטראקטיביות ואנימציות
- **Font Awesome 6** - אייקונים מקצועיים
- **Google Fonts** - גופן Assistant העברי

## 🚀 הפעלה מקומית

פשוט פתח את הקובץ `index.html` בדפדפן. הפרויקט לא דורש התקנה או שרת מקומי.

```bash
# אופציה 1: פתיחה ישירה
open index.html

# אופציה 2: עם שרת מקומי (אופציונלי)
python -m http.server 8000
# או
npx serve
```

## 🔧 התאמה אישית

### שינוי צבעים
ערוך את ה-CSS Variables ב-`:root` בקובץ `css/style.css`

### הוספת תוכן
כל הטקסטים נמצאים ב-`index.html` וניתנים לעריכה פשוטה

### שינוי מחירים
ערוך את הסקשן `.pricing-section` ב-HTML
